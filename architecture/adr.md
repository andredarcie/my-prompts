# Architecture Decision Record

Estrutura um ADR para documentar uma decisão técnica.

```text
Crie um ADR para a seguinte decisão:

[DESCREVA A DECISÃO]

Contexto do sistema:
[DESCREVA O CONTEXTO BREVEMENTE]

Inclua: contexto, opções consideradas (mínimo 3), decisão tomada, consequências positivas e negativas. Seja conciso.
```
